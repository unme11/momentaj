mfw-m1-ab_nrf9160_0.6.8-prealpha release notes

** Introduction to the mfw-m1-ab_nrf9160 release notes **

These release notes describe the changes in the mfw-m1_nrf9160 firmware from version to version.
The release notes are intended to list briefly all relevant changes in a given version.

*** mfw-m1-ab_nrf9160_0.6.8-131.prealpha
************************************

The mfw-m1 is the 3GPP LTE release 13 Cat-M1 firmware for the modem subsystem of nRF9160.

This firmware version is for nRF9160 Dev2 modules.

Operation is enabled on E-UTRA Bands 1, 2, 3, 4, 5, 8, 12, 13, 14, 17, 18, 19, 20, 25, 26, 27, 28 and 66.

This firmware is a band variant of mfw-m1_nrf9160_0.6.8-131.alpha. For details of 
functionality, limitations and known issues, see mfw-m1_nrf9160_0.6.8-alpha release notes.
